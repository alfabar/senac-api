Introdução

Para que serve esta API?

Visão geral

Esta é uma REST API que permite a manipulação de dados de alunos da nossa escola em formato JSON.

Códigos de erro de respostas


	
	
400: erro no acesso
	
	
	
404: recurso não encontrado
	
	
	
204: não há conteúdo
	
	
	
200: sucesso
	
	
	
201: novo recurso inserido
